Database x Windows Forms programm

Loodud Windows Formsi andmebaasi programm

Tegin programmi kasutades Microsoft Visual Studiot, .NET Frameworki ja C# programmeerimiskeelt. Programmi tegin selle juhendi järgi. See programm on Windows Formsi rakendus, mis kasutab Entity Framework Core andmebaasi haldamiseks. Programm võimaldab kasutajatel andmeid vaadata, lisada, muuta ja kustutada. Kasutasin andmebaasina juhendis välja toodud andmebaasi infot erinevate toiduainete tootekategooriate ja seal all olevate toitude kohta. Selle programmi tegemine aitas mul õppida andmebaasi ühendamist ja parandas minu oskuseid Visual Studio-ga töötamises. Allpool on näha ka loodud programmi ERD-diagramm ning lingid programmi koodideni.
